Easily create a golf tournament registration system

== Description ==

Throwing a golf tournament? This plugin is intended to help you plan, accept registrations for individuals and teams.




== Installation ==

1. Upload this plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
